This project helps people understand long, confusing documents.

For example: contracts, terms and conditions, rental agreements and such

A person pastes the text into the app.
The app then explains: what the document says, what parts are important and what parts may be risky

So instead of reading difficult legal language, the user gets a simpler explanation.

In this project, the AI helps by: reading the document, summarizing it, explaining it in simple words and answering questions about it

The stack used was:

- Next.js: this is used to build the website and the backend routes.

- React: this is used to build the page the user interacts with.

- TypeScript: this helps keep the code organized and reduce mistakes.

- Groq API: this is the AI service used to analyze the text and answer questions.

- Zod: this checks that the app is getting the right kind of input.
